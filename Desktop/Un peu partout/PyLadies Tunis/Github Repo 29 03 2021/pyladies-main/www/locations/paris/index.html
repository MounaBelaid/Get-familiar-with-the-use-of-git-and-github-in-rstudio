Introduction

Bonjour !

Nous sommes un groupe de mentoring qui a pour objectif d'aider les femmes (cis et trans), les hommes trans, les genderqueer ou les agenré⋅e⋅s à devenir des membres actifs et des leaders dans la communauté open-source Python.

Nous organisons plusieurs types d’évènements : des ateliers de programmation et des meetupus en non-mixité ainsi que des conférences où les hommes cis pourront venir uniquement en tant que +1 d'un⋅e membre du groupe.

N'hésitez pas à venir, quelque soit votre niveau en Python :)

Où nous trouver ?

  • Sur Twitter : @PyLadiesParis.
  • Sur Meetup : PyLadies-Paris.
  • Sur GitHub : PyLadies-Paris.

Code de Conduite

En plus du code de conduite définit par Pyladies, nous suivons un code de confuite renforcé construit à partir de celui du Geek Feminism Wiki. Vous le trouverez dans la partie forum de notre page meetup.
